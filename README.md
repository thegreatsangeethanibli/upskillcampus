# Air Quality Monitoring System

## Internship Project
Upskill Campus & UniConverge Technologies Pvt. Ltd.

## Description
This project is an IoT-based Air Quality Monitoring System using ESP32, DHT22, and MQ-2 sensors. It monitors temperature, humidity, gas levels, and calculates AQI. The data is sent to ThingSpeak cloud for real-time visualization.

## Technologies Used
- ESP32
- DHT22 Sensor
- MQ-2 Gas Sensor
- ThingSpeak
- Wokwi Simulator
